# Android Architecture Components
#### _Best Practices for building high-quality Android apps using the Android Architecture Components for data persistence and display_

The _Android Architecture Components_ are a set of open source libraries that were created by __Google__ to help<br />
developers create robust, maintainable apps for the Android platform. They were originally released in 2017<br />
at the Google I/O Conference and they've been used extensively in many commercial apps since then.
